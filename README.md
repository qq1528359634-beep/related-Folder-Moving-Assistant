# 関連フォルダ移動アシスタント (Related Folder Moving Assistant)

## 概要

**関連フォルダ移動アシスタント**は、Windows向けのWPFデスクトップアプリケーションです。
指定した移動元フォルダ内のサブフォルダを、移動先フォルダ内の**名前が一致するサブフォルダ**へ自動的に移動させることができます。同一ディスク間はもちろん、異なるディスク（ドライブ）間の移動にも対応しています。

---

## 主な機能

- **移動元・移動先フォルダの選択**  
  GUIのボタンからフォルダ選択ダイアログを開き、パスを設定できます。

- **フォルダ名による自動マッチング**  
  移動元フォルダ内のサブフォルダ名と、移動先フォルダ内のサブフォルダ名を比較し、一致するペアを自動検出します。

- **同一ディスク内の移動**  
  `Directory.Move` を使用した高速な移動を行います。

- **異なるディスク間の移動**  
  コピー＋削除方式による再帰的なフォルダ移動を行います（サブフォルダ・ファイルを含む）。

- **移動元フォルダ一覧の表示**  
  移動元を選択すると、含まれるサブフォルダの一覧がUIのリストビューに表示されます。

## 使い方

1. **移動先フォルダを選択する**  
   「Target Path」ボタンをクリックし、フォルダを選択します。  
   選択すると、そのフォルダ内のサブフォルダ一覧が内部に読み込まれます。

2. **移動元フォルダを選択する**  
   「Source Path」ボタンをクリックし、フォルダを選択します。  
   選択したフォルダのサブフォルダ一覧がUIのリストに表示されます。  
   ※ 移動先と同じパスは選択できません。

3. **「Move」ボタンを押して実行する**  
   移動元と移動先のフォルダ名が一致するペアを自動検出し、移動を実行します。  
   完了後、「Move Finished！」と表示されます。

---

## フォルダマッチングのロジック

```
移動元フォルダ（source）のサブフォルダ名
        ↕ 名前が一致するかチェック
移動先フォルダ（destination）のサブフォルダ名
```

- 移動先のサブフォルダ名が、移動元のサブフォルダ名に**含まれている**場合にマッチと判定します。
- マッチした移動先フォルダが**1つだけ**の場合のみ移動を実行します（曖昧なマッチは除外）。
- 移動先に同名フォルダが既に存在する場合、または移動元フォルダが見つからない場合は警告を表示してスキップします。

## プロジェクト構成

```
relatedFolderMovingAssistant/
├── Models/
│   └── Models.cs          # データモデル（FolderPath, SortedFolder）
├── Methods/
│   └── MyMethods.cs       # コアロジック（パス取得・マッチング・移動処理）
├── MainWindow.xaml        # UI レイアウト
└── MainWindow.xaml.cs     # UI イベントハンドラ
```

### 主要クラス

| クラス | 役割 |
|--------|------|
| `FolderPath` | 移動元・移動先のパスとフォルダ一覧を保持する静的クラス |
| `SortedFolder` | マッチング済みのフォルダペア（移動元・移動先）を表すモデル |
| `MyMethods` | フォルダ選択・マッチング・移動処理などのロジックをまとめたクラス |

---

## 注意事項

- 移動元と移動先に**同じパス**は指定できません。
- 異なるディスク間の移動はコピー後に元フォルダを削除する方式のため、移動中にエラーが発生した場合、**元データが一部残る可能性**があります。
- 移動先サブフォルダへのマッチが**2つ以上**ある場合、そのフォルダは移動対象から除外されます。
