# Related Folder Moving Assistant (関連フォルダ一括移動アシスタント)

WPF (.NET) で開発された、関連性のあるフォルダを自動でマッチング、効率的に一括移動するためのデスクトップアプリケーションです。

---
主な機能

*   **直感的なパス選択**：移動元（Source）と移動先（Destination）のフォルダパスをボタン一つで簡単に選択可能。
*   **安全なバリデーション**：空のパスや、移動元と移動先が同一である場合の誤操作を自動で検知・ブロック。
*   **フォルダの一覧表示**：移動元のフォルダ群を自動で読み込み、画面上の `ListView` にインデックス番号（ID）付きで分かりやすく表示。
*   **インテリジェントな関連移動**：
    *   移動元と移動先のフォルダ構成を解析し、関連性のあるフォルダを抽出。
    *   移動時の競合や依存関係を防ぐため、フォルダの最適な移動順序を自動でソート。
    *   ボタン一つで一括移動を実行し、完了時にポップアップで通知。

## 🚀 核心ロジックの概要

本ツールの主要な処理は以下の3つのイベントで構成されています。

1.  **移動先パスの設定 (`SelectMonPath_Click`)**
    *   選択された移動先パスを `FolderPath.destiPath` に格納し、配下のフォルダ構造を事前に読み込みます。

2.  **移動元パスの設定と一覧表示 (`SelectSonPath_Click`)**
    *   移動元パスの整合性をチェック（空チェック、移動先との重複チェック）。
    *   LINQ の `.Select((e, Index) => ...)` を活用し、読み込んだフォルダ一覧に 1 から始まる連番（ID）を動的に付与して `ItemsSource` にバインド、画面に表示します。

3.  **フォルダ移動の実行 (`Move_Click`)**
    *   両方のパスが正しく設定されているか安全確認を行います。
    *   独自のアルゴリズムである `GetRelevantFolders()`（関連抽出） $\rightarrow$ `SortOneRelevantFolders()`（ソート処理） $\rightarrow$ `ExecuteMoveFolder()`（移動実行）を順に呼び出し、安全に処理を完了させます。
